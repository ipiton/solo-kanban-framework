# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1, as its Code of Conduct.

The full text is available at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

## Scope

The Code of Conduct applies in all project spaces — issues, pull requests, discussions, and any other communication channel associated with this repository — and when an individual is officially representing the project in public spaces.

## Reporting

To report a concern, contact the maintainer by opening a confidential issue or by email at `semenovvitaliy@icloud.com`. All reports will be reviewed and handled with discretion.

## Enforcement

The maintainer is responsible for clarifying the standards and is expected to take fair corrective action in response to any behavior they deem inappropriate. The maintainer may remove comments, commits, code, issues, and other contributions that are not aligned with the Code of Conduct, and may temporarily or permanently ban any contributor for behavior they deem inappropriate.
