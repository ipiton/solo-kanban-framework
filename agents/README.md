# Agent Adapters

This directory contains starter instructions for AI coding agents. They are intentionally generic. Copy and adapt them inside a real repository instead of treating them as a complete automation layer.

Recommended layering:

1. Keep the Solo Kanban method and artifacts stable.
2. Add repository-specific engineering rules locally.
3. Add tool-specific commands or skills as adapters.
4. Keep deployment, infrastructure, and product-specific rules outside the core framework.
