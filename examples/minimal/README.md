# Minimal Solo Kanban Project

A minimal project using Solo Kanban needs only planning files and task templates.

```text
project/
  docs/planning/NEXT.md
  docs/planning/DONE.md
  docs/planning/BUGS.md
  docs/planning/TECH-DEBT.md
  docs/planning/BACKLOG.md
  docs/planning/DECISIONS.md
  tasks/templates/requirements.md
  tasks/templates/research-light.md
  tasks/templates/research.md
  tasks/templates/Spec.md
  tasks/templates/tasks.md
  tasks/archive/
```

Start with one queued task in `NEXT.md`, create `tasks/<slug>/requirements.md`, and keep all task work in that workspace until it is archived.
